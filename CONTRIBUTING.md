# Contributing

Grazie per l'interesse verso `high-cde/zdos-zlang`. Il repository fa parte dell'ecosistema ZDOS / Z-GENESIS e privilegia contributi chiari, verificabili e coerenti con la documentazione.

## Flusso consigliato

| Passaggio | Descrizione |
|---|---|
| Fork o branch | Lavora su un branch dedicato e con nome descrittivo. |
| Modifica mirata | Mantieni le modifiche piccole e facili da revisionare. |
| Documentazione | Aggiorna README, `docs/` o wiki quando cambia il comportamento del progetto. |
| Test | Esegui i test disponibili o descrivi le verifiche manuali effettuate. |
| Pull request | Spiega motivazione, impatto e possibili rischi della modifica. |

## Stile

Usa un linguaggio tecnico, diretto e professionale. Evita di introdurre dipendenze non necessarie e non includere credenziali o dati sensibili nei commit.
