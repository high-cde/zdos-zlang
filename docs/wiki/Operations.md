# Guida operativa

## Avvio

Clona il repository e consulta il README per i comandi più adatti alla tecnologia rilevata. Prima dell'esecuzione, verifica sempre dipendenze, permessi e configurazioni locali.

## Manutenzione

| Attività | Frequenza consigliata |
|---|---|
| Verifica dipendenze | A ogni modifica significativa. |
| Aggiornamento documentazione | A ogni feature o cambio architetturale. |
| Controllo segreti | Prima di ogni push. |
| Revisione roadmap | Periodicamente, in base all'evoluzione del progetto. |

## Sicurezza

Non eseguire automazioni su sistemi non autorizzati. Conserva credenziali e token fuori dal repository e usa ambienti isolati per testare script sperimentali.
