# Specifica delle Syscall ZDOS per ZLang

## 4.4. Syscall ZDOS

Tabella di syscall (esempio):

- `SYS_LOG` → `sys.log(tag, msg)`
- `SYS_EXEC` → `sys.exec(cmd, args)`
- `SYSEXECCAPTURE`
- `SYSREGGET` → `sys.registry.get(key)`
- `SYSREGSET`
- `SYSEVENTEMIT`
- `SYSTIMENOW`
- `SYSNETCONNECT`, `SYSNETSEND`, `SYSNETRECV`
